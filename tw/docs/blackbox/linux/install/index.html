在 Linux 上安裝黑盒子

第一步：安裝 Docker

黑盒子是以 Docker Image 的方式發行，在安裝黑盒子前，系統裡必須先安裝 Docker。不同的 Linux 版本，安裝 Docker 的方法都不盡相同，請參考 Docker 官網 Install Docker Engine 安裝說明。以 Ubuntu 16.04 為例，安裝 Docker 最簡單的方法是：

$ sudo apt-get install docker.io

第二步：安裝黑盒子

用下面指令來下載 1Album 黑盒子：

$ docker pull 1album/blackbox

第三步：第一次啟動

第一次啟動黑盒子，需要用以下的指令來設定環境：

$ docker run -itd \
         -v /path/to/data:/data \
         --name mybox \
         1album/blackbox

其中最重要的是 -v 參數，用來設定資料目錄。資料目錄 /data 是 1Album 黑盒子儲存所有資料的位置，必須將 /data 指向電腦的檔案夾才能啟動黑盒子，上面的例子將 /data 指向電腦裡的 /path/to/data。您可以選擇任意檔案夾，但這個檔案夾內的資料必須由黑盒子全權控制，不要手動改變內容，否則可能會造成資料遺失或錯誤。

除了 /data 外，還有三個額外的目錄可以設定，這些沒設也沒關係。/export 用來將黑盒子的內部資料輸出成一般檔案，/import 用來輸入外部資料進黑盒子，而 /cache 用來儲存如照片縮小圖等的暫存檔。下面的例子，把所有目錄都指向電腦的檔案夾：

$ docker run -itd \
         -v /path/to/data:/data \
         -v /path/to/cache:/cache \
         -v /path/to/import:/import \
         -v /path/to/export:/export \
         --name mybox \
         1album/blackbox

執行 docker run 後，可用 docker ps 來查看執行中的 Docker containers。最後一欄顯示的是以 --name 參數指定的名字。

第四步：連結帳號

必須把黑盒子加到您的帳號中，才能開始使用。黑盒子的 Device Key 可以從 logs 中找到。下面的 mybox 是先前以 --name 參數指定的名字：

$ docker logs mybox
