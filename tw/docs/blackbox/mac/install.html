在 OSX 上安裝黑盒子

第一步：安裝 Docker

黑盒子是以 Docker Image 的方式發行，在安裝黑盒子前，系統裡必須先安裝 Docker。點 這裡下載 Docker Toolbox ，詳細安裝說明請參考 Docker Toolbox 安裝說明。

裝好後，在應用程式及 LaunchPad 裡會多出兩個程式，打開箭頭指的那個 Kitematic：

啟動後會看到下面這個登入的畫面，不登入沒有關係，直接按畫面右下角 SKIP FOR NOW 跳過：

離開登入頁面後，進入 Kitematic 的首頁：

第二步：安裝黑盒子

在搜尋框裡輸入 1album 來搜尋 1Album 黑盒子程式，找到後按右下角的 CREATE 鈕來下載並安裝。

第三步：第一次啟動

資料目錄 /data 是 1Album 黑盒子儲存所有資料的位置，必須將 /data 指向電腦的檔案夾才能啟動黑盒子。如下圖，點 Settings -> Volumes 到設定頁面，點對應 /data 的 CHANGE 紐：

按下 CHANGE 後跳出對話框選擇檔案夾。這個檔案夾內的資料必須由黑盒子全權控制，請不要手動改變內容，否則可能會造成資料遺失或錯誤。

選定後如下圖，可以看出 Docker 內 /data 目錄所對應的電腦檔案夾。

除了 /data 外，還有三個額外的目錄可以設定，這些沒設也沒關係。/export 用來將黑盒子的內部資料輸出成一般檔案，/import 用來輸入外部資料進黑盒子，而 /cache 用來儲存如照片縮小圖等的暫存檔。

設定好目錄，點選下圖的 HOME，會看到黑盒子已經啟動完成了：
