Install Blackbox On Synology NAS

Step 1: Install Docker

1Album Blackbox is distributed as a Docker Image. You must install Docker on your Synology NAS before installing blackbox. Many Synology NAS models support this, but it seems limited to those with a “+” symbol at the end of its model number. For a full list of supported models, please check the bottom of the Synology Docker Package page.

To install Docker, login to your Synology NAS, open Package Center on the desktop and search for Docker, then click the Install button. If you cannot find Docker, your model does not support it.

When installation is done, Docker is listed in the installed packages:

Docker is also listed in the main menu. Click to open it:

This is the main page of Docker on Synology NAS:

Step 2: Install Blackbox

Switch to Registry tab on the left, then enter 1album in the search box to search for 1Album Blackbox.

Selected the item then click Download button to download:

When download is in progress, the icon on the right is animated:

When download is completed, the icon will stop animating, and a notification message will be posted:

Step 3: First Launch

Switch to the ‘Image’ tab on the left, select 1Album Blackbox image, then click Launch button to launch it:

A settings dialog shows up. Click Advanced Settings in the dialog box:

The Enable auto restart function circled below can restart the blacbox automatically should it encounter an unexpected error and quite. We recommand you to enable it:

Now switch to the Volume tab. 1Album Blackbox stores everything in /data volume. You must associate it to a local folder before starting the blackbox. You do that by click the Add Folder button below:

Select any folder you want, or click Create Folder button to create a new one. Note that you must not manually alter the content in this folder, otherwise your data inside the blackbox may be damaged or lost. Click the Select button when you’re done.

After a local folder is selected, enter /data (all lowercase) in the Mount path field. Now the /data volume is configured.

Besides /data, there are three optional volumes that can be configured. /export is for exporting data out of the blackbox, and /import is for importing data back in. /cache is for storing temporarily files such as thumbnail images. In the example below, each of the volumes is linked to a local folder in NAS. Click OK when you’re done:

We are back to the settings dialog. Click Next:

Make sure at least the /data volume is linked to a local folder, then click Apply:

Step 4: Link Account

You must add the blackbox to your account to start using it. First switch to Container tab on the left, select the running blackbox, then click the Details button:

Switch to the Log page in the detail dialog. You can find the Device Key inside:

Then open 1Album app and enter the Device Key as below:

Now this blackbox will work for you. Your account name is also printed in the blackbox log. This log page does not refresh automatically, you need to click the little refresh button at the lower right corner to refresh:
