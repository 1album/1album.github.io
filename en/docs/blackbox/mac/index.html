Install On Mac OSX

Step 1: Install Docker

1Album Blackbox is distributed as a Docker Image. You must have Docker installed on your system to use it. Click here to download Docker Toolbox . Please reference Docker Toolbox installation guide for installation.

After installation, click the Kitematic icon to start Docker:

A login page shows up. Login is optional, you can just click the SKIP FOR NOW button:

After the login page, Kitematic home page shows up:

Step 2: Install Blackbox

Enter 1album in the search box to search for 1Album Blackbox, then click the CREATE button to download and install it:

Downloading:

After download is completed, this error message shows up. Don’t worry, this is because we haven’t setup /data volume yet.

Step 3: Setup Data Volunme

1Album Blackbox stores everything in /data volume. You must associate it to a local folder before starting the blackbox. Open Seetings -> Volumes page, then click the CHANGE button corresponding to the /data volume:

Select any folder you want. Note that you must not manually alter the content in this folder, otherwise your data inside the blackbox may be damaged or lost.

After a folder is selected, it is listed next to the /data volume:

Besides /data, there are three optional volumes that can be configured. /export is for exporting data out of the blackbox, and /import is for importing data back in. /cache is for storing temporarily files such as thumbnail images.

Click HOME when you’re done. The blackbox is now started:

Step 4: Link Account

You must add the blackbox to your account to start using it. First, write down the Device Key:

Then open 1Album app and enter the Device Key as below:

Now this blackbox will work for you. Your account name is listed on the blackbox console:
