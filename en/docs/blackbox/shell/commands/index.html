Blackbox Commands

In the command shell, you can type in help to get a list of available commands:

blackbox> help
Available commands:

   export       export library content to an external folder
   import       unport library content from an external folder
   netstat      print network status
   resolver     print the status of all peers associated with this account
   start        start blackbox manually
   status       print system status
   stop         stop blackbox manually
   sync         sync this blackbox with primary blackbox
   unbind       unbind this blackbox from your account
   version      print version information
   shutdown     shutdown blackbox and exit the program

blackbox>

You can use help [command] to get detail information about a command. For example:

blackbox> help export
export: 
usage: export [--src src_path] dest_folder

       src_path: absolute source path in your 1Album repository, default to /Library
    dest_folder: destination folder name under the export volume, no default value
        
examples:
       export my_dest
            Export everything to /export/my_dest
        
       export --src /Library my_dest
            Same as above, export everything to /export/my_dest
        
       export --src /Library/Timeline/2016/03 my_dest
            Export all March, 2016 files under timeline to /export/my_dest
        
       export --src /Library/Folders/Foo my_dest
            Export Foo folder to /export/my_dest
        
       export --src /Albums/my_trip my_dest
            Export my_trip album to /export/my_dest

blackbox>
