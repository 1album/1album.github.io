Enter Blackbox Shell

1Album Blackbox is designed to run autonomously, so entering its command shell is not necessary. Nonetheless, there are advanced features that are available only through the command shell. The following describes how to enter the command shell on each of the operating systems. For a list of commands available in the shell, please reference the Blackbox Commands page.

Mac OSX

First please make sure that the blackbox is running. The command shell is available only when the blackbox is running. Now click the DOCKER CLI button in the lower left corner:

The OSX terminal window pops up. Enter docker attach blackbox then hit Enter twice. Once you see the blackbox> prompt, you are in the command shell. If there are multiple blackboxes installed, you need to put the name shown in the Kitematic interface after attach.

Hit Ctrl-P then Ctrl-Q to detach from the command shell and back to OSX shell. You can also close the terminal window directly without hitting Ctrl-P Ctrl-Q, the shell will be detached automatically.

Windows

First please make sure that the blackbox is running. The command shell is available only when the blackbox is running. Now click the DOCKER CLI button in the lower left corner:

The Windows terminal window pops up. Enter docker attach blackbox then hit Enter twice. Once you see the blackbox> prompt, you are in the command shell. If there are multiple blackboxes installed, you need to put the name shown in the Kitematic interface after attach.

Hit Ctrl-P then Ctrl-Q to detach from the command shell and back to Windows shell. You can also close the terminal window directly without hitting Ctrl-P Ctrl-Q, the shell will be detached automatically.

Linux

Use the following command to enter the blackbox shell. You usually need to press Enter twice to see the blackbox> prompt. Once you see the prompt, you are in the shell. mybox below is what you specified with the --name argument when you setup this blackbox:

$ docker attach mybox

Hit Ctrl-P then Ctrl-Q to detach from the command shell and back to Linux shell. You can also close the terminal window directly without hitting Ctrl-P Ctrl-Q, the shell will be detached automatically.

Synology NAS

First please make sure that the blackbox is running. The command shell is available only when the blackbox is running. Now click the Details button:

The details dialog pops up. Switch to the Terminal tab, and hit Enter in the black area. Once you see the blackbox> prompt, you are in the blackbox shell. Note that the current Safari browser on OSX cannot open this terminal page. Please use Chrome or Firefox on OSX instead. Windows browsers work fine.

Hit Ctrl-P then Ctrl-Q to detach from the command shell. The black area becomes gray, and you need to re-open this dialog box if you want to enter the command shell again. You can also close the dialog window directly without hitting Ctrl-P Ctrl-Q, the shell will be detached automatically.
